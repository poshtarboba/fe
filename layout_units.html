Одиниці вимірювання, формат кольору

Розміри можна задавати в пікселях, типографських точках (поінтах), метричних та імперських одиницях, відсотках.

height: 20px;
width: 50%;
font-size: 16pt;
line-height: 1.3em;
padding: 1cm 0.2inc 0;

Висоту у відсотках для елементів можна задавати тільки якщо вони спозиціоновані абсолютно чи фіксовано.

Після нуля одиниці виміру не ставляться.

Відсотки від розміру вікна браузера (viewport width, viewport height):

width: 80vw;
height: 50vh;

Кутові одиниці виміру:

transform: rotate(360deg); // градуси
transform: rotate(250grad); // 360 градусів = 400grad, прямий кут = 100grad
transform: rotate(3.14rad); // радіани
transform: rotate(0.5turn); // 1 turn = 360 градусів

Колір можна задавати чотирма способами:

color: green;
border-color: #333aaa;
background: rgb(127, 255, 0);
background-color: rgba(255, 0, 160, 0.3);

Використовуйте маленькі літери.

Якщо при заданні кольора в шіснадцятковому вигляді пари цифр/букв повторюються - колір бажано записати у скороченому виді, трьома символами:

color: #33ff99;
color: #3f9;
