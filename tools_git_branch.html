Гілки Git

Створити нову гілку і перейти в неї:

git checkout -b назва_гілки

Перейти в існуючу гілку:

git checkout назва_гілки

Вивести список гілок:

git branch

Заляти гілку на сервер:

git push origin назва_гілки

Об'єднання гілок:

git merge назва_гілки

Завдання.

  1. Створіть гілку test1, в html-файл пропишіть текст "test 1".
  2. Перейдіть в головну гілку, створіть гілку test2, зробіть текст жовтого кольору.
  3. Створіть нову гілку test3 від гілки test2, поміняйте розмір шрифта на 24px.
  4. Перейдіть в гілку test2, об'єднайте її з гілкою test1 та результат заляйте на git.
