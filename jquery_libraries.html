Бібліотеки

Bootstrap

Фреймворк Bootstrap містить в собі не тільки сітку та візуальні компоненти, але і бібліотеки JS.

JS-компоненти розділені по окремим js-файлам, що дозволяє вибірково підключати бібліотеки та економити кілобайти.

jQuery UI

Велика бібліотека компонентів jQuery UI містить багато різних віджетів.

Є можливість скомпілювати бібліотеку лише з тими компонентами, які вам потрібні в проекті.

Завдання.

  1. Підключіть в проект повну бібліотеку jQuery UI.
  2. Добавте на сторінку компоненти Droppable і Sortable.
  3. Добавте на сторінку поле вводу дати і Datepicker для нього.
  4. Добавте кнопку і відображайте popup (Dialog) при кліку на кнопку.

Parallax

Розгляньте можливості бібліотек, що допомагають створювати параллакс-ефекти.

Домашнє завдання

  1. Підключіть до проекту bootstrap, зробіть щоб при кліку на кнопку викликався popup (modals).
  2. Добавте до кнопки підказку (tooltips).
  3. Добавте на сторінку акордион (collapse) та слайдер (carousel) з бібліотеки bootstrap.
  4. Дослідіть бібліотеку MooTools.
