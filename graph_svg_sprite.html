SVG-спрайти

Вставити куди-небудь у веб-сторінку SVG-спрайт і сховати його.

Кожна іконка огортається в тег <symbol> з ідентифікатором і viewBox:

<svg ... style="display: none;">
  <symbol id="iconFb" viewBox="0 0 24 28">
    <ellipse ... class="fb-ellipse" />
    <path ... />
    <path ... />
  </symbol>
  <symbol id="iconTw" viewBox="0 0 25 26">
    <path ... />
  </symbol>
</svg>

Для вставки іконок використовуємо тег <use>:

<svg width="60" height="60">
  <use xlink:href="#iconFb" />
</svg>
<svg width="24" height="24" class="fb-small">
  <use xlink:href="#iconFb" />
</svg>
<svg width="24" height="24">
  <use xlink:href="#iconTw" />
</svg>

Можна використовувати CSS для зміни кольорів заливки, ліній і т.п. як всієї іконки, так і окремо примітивів:

.fb-small { fill: blue; }
.fb-ellipse { stroke: red; }
